> **SKILLS**
> 

### **Python**

Python을 사용하여 데이터 처리, 분석, 웹 개발을 수행한 경험이 있습니다.

Pandas, SQLAlchemy, Django ORM을 활용하여 데이터 파이프라인을 구축할 수 있습니다.

구조화된 코드를 작성하여 유지보수가 용이한 프로그램을 개발할 수 있습니다.

---

### **Backend**

Django 프레임워크를 기반으로 웹 서버를 구축하고 운영한 경험이 있습니다.

ORM, API 설계, 데이터베이스 관리, 배포 과정을 이해하고 있습니다.

PostgreSQL과 MariaDB를 활용하여 효율적인 쿼리를 작성할 수 있습니다.

---

### **Frontend (HTML / CSS / JavaScript)**

Django 템플릿과 JavaScript를 활용하여 웹 UI를 구현한 경험이 있습니다.

HTML5, CSS3를 사용하여 반응형 디자인을 적용할 수 있습니다.

Ajax와 Fetch를 이용하여 비동기 통신을 구현할 수 있습니다.

---

### **Data Analysis**

Python을 사용하여 데이터 정제와 분석을 수행할 수 있습니다.

SQL 및 Python을 활용하여  통계 분석을 수행한 경험이 있습니다.

분석 결과를 시각화하여 인사이트를 도출할 수 있습니다.

---

### **DevOps / Deployment**

Render환경에 Django 애플리케이션을 배포한 경험이 있습니다.

Git과 GitHub를 사용하여 프로젝트 버전 관리 및 협업을 할 수 있습니다.

---

### **Database**

PostgreSQL과 MariaDB를 사용하여 테이블을 설계하고 관리할 수 있습니다.

Django ORM을 이용하여 효율적인 데이터 접근 로직을 구현할 수 있습니다.
